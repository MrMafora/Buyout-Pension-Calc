## Keyword Usage Guidelines

### Title Tag (50-60 characters)
- Include primary keyword near beginning
- Example: "VSIP for Federal Employees: Complete 2024 Guide"

### First Paragraph
- Use primary keyword within first 100 words
- Include one secondary keyword naturally

### Headers (H2, H3)
- Include keywords in 1-2 headers
- Use question format for featured snippet opportunities

### Meta Description (150-160 characters)
- Include primary keyword
- Add compelling call-to-action
- Example: "Learn everything about VSIP for federal employees including eligibility, payment calculations, and whether you should take the buyout."

### Content Body
- Keyword density: 1-2% (natural usage)
- Use variations and synonyms
- Include LSI keywords (related terms)

### URL Slug
- Primary keyword, hyphen-separated
- Keep under 60 characters
- Example: `vsip-federal-employee-guide`

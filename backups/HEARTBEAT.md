# HEARTBEAT.md

Check the following periodically (e.g., every 1-2 hours):

1. **Leads:** Run `lead-tracker` scripts to check for new website form submissions.
2. **Email:** Check `support@fedbuyout.com` for urgent inquiries.
3. **News:** Run `news-monitor` to scan for major federal buyout news.
4. **System:** Run `health-checker` to verify disk space and service status.

**Rules:**
- If nothing new/urgent found: Reply `HEARTBEAT_OK`
- If new leads/emails/alerts: Send a brief summary message
- Don't repeat alerts for the same item
